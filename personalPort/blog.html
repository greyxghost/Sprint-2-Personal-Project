First Blog Post

Dec 3rd, 2024

As a college student passionate about web design and development, I’m excited to use this space to document my journey, share my work, and connect with others in the field. Every day, I’m learning something new, and this blog is my way of reflecting on that growth while showcasing the projects I’ve been working on.

Here are a few of the projects I’ve completed so far:

  • Major City Weather App: A weather application that fetches real-time weather data using a city name, built with an API integration. This project taught me a lot about working with external data and creating functional, user-friendly designs.
  • Basic Python Calculator: A simple yet effective calculator built in Python that performs basic arithmetic operations. I added features like error handling to ensure smooth usability.
  • Snake Game: A recreation of the classic snake game, built with JavaScript, HTML, and CSS. It was a fun way to work on game mechanics and bring a nostalgic concept to life.

I’ll be uploading these projects to my GitHub soon, so you can explore the code and see how they work. This journey has been incredibly rewarding, and I look forward to continuing to refine my skills and create even more meaningful projects. Thank you for stopping by—I’m excited to share more with you as I continue to grow in this field.
